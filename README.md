# better-react-transliterate workspace

This repository is now a `pnpm` workspace with:

- `packages/better-react-transliterate`: the publishable SSR-friendly React transliteration library
- `examples/vite-react`: a Vite + React + TypeScript example wired to the local package

## Quick start

```bash
pnpm install
pnpm build:library
pnpm dev
```

## Workspace commands

```bash
pnpm build
pnpm build:library
pnpm build:example
pnpm test
pnpm typecheck
```
